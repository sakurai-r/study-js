```ts
for (i = 1; i < 101; i++)
  console.log((i % 3 ? "" : "Fizz") + (i % 5 ? "" : "Buzz") || i);
```

"Fizz" の場合

```
i % 3 ? "" : "Fizz" の判定では、3で割り切れるのであまりが0となり、三項演算子の判定は false になる。 -> "Fizz"
i % 5 ? "" : "Buzz" の判定では、5で割り切れないのであまりが0以外となり、三項演算子の判定は true になる。 -> ""

したがって "Fizz" || i となり "Fizz" は true に変換される値のため "Fizz" だけが出力される。
```

"Buzz" の場合

```
i % 3 ? "" : "Fizz" の判定では、3で割り切れないのであまりが0以外となり、三項演算子の判定は true になる。 -> ""
i % 5 ? "" : "Buzz" の判定では、5で割り切れるのであまりが0となり、三項演算子の判定は false になる。 -> "Buzz"

したがって "Buzz" || i となり "Buzz" は true に変換される値のため "Buzz" が出力される。
```
