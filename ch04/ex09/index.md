### undefined

```
type of undefined -> "undefined"
```

### null

```
type of null -> "object"
```

nullとオブジェクトを区別したい場合は、明示的に null かどうかテストする

### オブジェクト

```
type of オブジェクト -> "object"
```

オペランドが関数以外のオブジェクトと配列の場合、typeof 演算子は "object" としか返さない

### NaN

```
type of NaN -> "number"
```

### 数値

```
type of 数値 -> "number"
```

### 関数

```
type of 関数 -> "function"
```
